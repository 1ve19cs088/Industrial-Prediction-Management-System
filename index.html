How It Works — Random Forest Algorithm
// Machine Learning pipeline overview
STEP 1 — DATA INPUT
Enter company metrics: employees, salary, production units, revenue, expenses, attendance & satisfaction scores.
STEP 2 — FEATURE SCALING
StandardScaler normalizes all features to mean=0, std=1 for optimal Random Forest performance.
STEP 3 — RF PREDICTION
3 Random Forest models (100 trees each) predict workforce, profit/loss, and resignation risk independently.
STEP 4 — RESULTS
Predictions are returned with confidence scores and visualized as charts and actionable reports.

Prediction Insights
// What each prediction means for your industry
👷 Workforce Prediction
Estimates the optimal number of workers required based on production targets, current headcount, attendance, and historical demand patterns. Helps avoid over/under-staffing.
💰 Profit / Loss Prediction
Analyzes the financial health of the organization by comparing revenue against operational costs. Uses Random Forest classification to predict profitable vs loss-making periods.
🚪 Resignation Risk Prediction
Identifies employee attrition risk based on satisfaction scores and attendance patterns. Helps HR proactively retain talent before resignations occur.

🎯 Project Objectives

▸

Workforce Prediction — Estimate optimal number of workers needed based on production and attendance data.

▸

Profit/Loss Estimation — Forecast financial outcomes using revenue vs expense analysis.

▸

Resignation Risk — Identify high-risk employees based on satisfaction and attendance patterns.

▸

Better Decision Making — Provide managers with AI-powered insights for smarter planning.

▸

Reduce Manual Work — Automate calculations replacing error-prone Excel-based methods.

🛠️ Technology Stack

▸

Frontend: HTML5, CSS3, JavaScript (ES6+), Chart.js

▸

Backend: Python 3.x, Flask (Web Framework)

▸

ML Engine: Scikit-learn — Random Forest (Classifier + Regressor)

▸

Data Processing: Pandas, NumPy, StandardScaler

▸

Database: SQLite3 (lightweight, file-based)

▸

Dev Tool: VS Code | Browser: Chrome/Edge

🔧 System Modules

▸

User Login Module — Secure SHA-256 hashed authentication for managers and admins.

▸

Data Entry Module — Structured form to input 7 key industrial metrics.

▸

Data Processing Module — Feature scaling + Random Forest ML pipeline.

▸

Prediction Module — 3 separate RF models for each prediction type.

▸

Visualization Module — Interactive Chart.js charts for results and analytics.

▸

History Module — SQLite-backed prediction log with full input/output records.
